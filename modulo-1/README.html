Ejemplos en Consola AWS - Módulo 1

Catálogo de Vinilos Serverless

📖 Índice de Guías

Este directorio contiene las guías paso a paso para construir una API serverless completa usando la consola de AWS.

Orden de Ejecución

# Guía Duración Servicios
0 Visión General 5 min Introducción
1 Lambda Hardcoded 15 min Lambda
2 Añadir DynamoDB 20 min Lambda + DynamoDB + IAM
3 Añadir S3 Imágenes 20 min S3 + Bucket Policies
4 Exponer con API Gateway 15 min API Gateway + CORS

Tiempo total: ~75 minutos (1 hora 15 min)

🎯 Objetivo Final

Construir esta arquitectura serverless:

Cliente (curl/navegador)
    ↓
API Gateway (GET /vinilos)
    ↓
Lambda (get-vinilos)
    ↓
DynamoDB (tabla vinilos) + S3 (imágenes)
    ↓
Respuesta JSON con catálogo

📚 Recursos Creados

Al finalizar las 4 iteraciones, habrás creado:

Recurso Nombre Tipo
Lambda {prefijo}-get-vinilos Función Node.js 20.x
DynamoDB {prefijo}-vinilos Tabla NoSQL
S3 Bucket {prefijo}-vinilos-imagenes Almacenamiento objetos
API Gateway {prefijo}-vinilos-api HTTP API
IAM Role {prefijo}-get-vinilos-role-* Permisos Lambda

Total de servicios AWS: 5

🎓 Conceptos Aprendidos

Servicios AWS

  • ✅ AWS Lambda: Computación serverless
  • ✅ Amazon DynamoDB: Base de datos NoSQL
  • ✅ Amazon S3: Almacenamiento de objetos
  • ✅ Amazon API Gateway: APIs HTTP
  • ✅ AWS IAM: Gestión de permisos
  • ✅ Amazon CloudWatch: Logs y monitorización

Conceptos Serverless

  • ✅ Arquitectura event-driven
  • ✅ Pay-per-use (pago por uso)
  • ✅ Escalado automático
  • ✅ Sin gestión de servidores
  • ✅ Alta disponibilidad
  • ✅ Integración de servicios

Conceptos de Desarrollo

  • ✅ APIs REST
  • ✅ JSON como formato de intercambio
  • ✅ Códigos de estado HTTP
  • ✅ CORS (Cross-Origin Resource Sharing)
  • ✅ Manejo de errores
  • ✅ Logging y debugging

🐛 Soporte

Si encuentras problemas:

  1. Revisa el troubleshooting de cada guía
  2. Verifica la región: debe ser eu-west-1 (Irlanda)
  3. Revisa CloudWatch Logs: Busca errores en los logs de Lambda
  4. Comprueba IAM: Verifica que Lambda tiene los permisos necesarios
